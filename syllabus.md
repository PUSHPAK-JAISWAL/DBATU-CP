# Course Syllabus: Competitive Programming and Algorithms

## Unit I: Introduction (7 hrs)
* **Core Concepts:** Online Judge (Programming Challenges Robot Judge), Understanding Feedback, Language Selection, Standard I/O, Programming Hints, Elementary Data Types.
* **Challenging Problems:**
    1. The $3n + 1$ Problem
    2. Minesweeper
    3. The Trip
    4. LCD Display
    5. Graphical Editor
    6. Interpreter
    7. Check the Check
    8. Australian Voting

---

## Unit II: Sorting (6 hrs)
* **Core Concepts:** Sorting Applications, Sorting Algorithms, Program Design (Rating the Field), Sorting Library Functions.
* **Challenging Problems:**
    1. Vito’s Family
    2. Stacks of Flapjacks
    3. Bridge
    4. Longest Nap
    5. Shoemaker’s Problem
    6. CDVII
    7. Shell Sort
    8. Football

---

## Unit III: Number Theory (5 hrs)
* **Primes:** Finding Primes, Counting Primes.
* **Divisibility:** Greatest Common Divisor (GCD), Least Common Multiple (LCM).
* **Modular Arithmetic:** Congruences, Operations on Congruences, Solving Linear Congruences.
* **Advanced Topics:** Diophantine Equations, Number Theoretic Libraries.

---

## Unit IV: Backtracking and Dynamic Programming (7 hrs)
* **Backtracking:** Constructing Subsets and Permutations, The Eight-Queens Problem, Pruning Search.
* **Dynamic Programming:** Introduction, Characteristics and Components, Comparison with Divide-and-Conquer.
* **Challenging Problems:**
    1. Little Bishops
    2. 15-Puzzle Problem
    3. Queue
    4. Servicing Stations
    5. Tug of War
    6. Garden of Eden
    7. Colour Hash
    8. Bigger Square Please

---

## Unit V: Graph Traversal (7 hrs)
* **Fundamentals:** Flavors of Graphs, Data Structures for Graphs.
* **Traversal:** Breadth-First Search (BFS), Finding Paths, Depth-First Search (DFS), Finding Cycles, Connected Components, Topological Sorting.
* **Advanced Graph Theory:** Degree Properties, Connectivity, Planar Graphs, Minimum Spanning Trees (MST).
* **Algorithms:** Dijkstra’s Algorithm, All-Pairs Shortest Path, Network Flows.
